# eXTC
Barebones Python based Twitter client. Created for use with [eXF1LT](https://github.com/eXhumer/pyeXF1LT).

## Package Installation Guide
### Requires
* Python 3.9+
* pip package manager

### Install from source
```console
pip install .
```

### Install from PyPI
```console
pip install extc
```
